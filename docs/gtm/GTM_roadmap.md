# AICO GTM Roadmap

This roadmap tracks concrete tasks to execute the GTM strategy. Use it as a living document.

## Phase 1 – Foundation (Weeks 1–2)

- [ ] **Website: AICO landing page (Squarespace)**
  - [ ] Create `/aico` page on https://boeni.industries
  - [ ] Add hero section (AICO positioning + primary CTA buttons)
  - [ ] Add sections: What AICO is, Product Line (Core / Pro / Enterprise), Features, FAQ
  - [ ] Add CTAs: "Open Source on GitHub" and "Contact us about AICO Pro / Enterprise"
  - [ ] Link to docs (GitHub Pages) and community (Discord, GitHub Discussions)

- [ ] **GitHub: polishing for public launch**
  - [ ] Add "Start here" section to `README.md` (users + contributors)
  - [ ] Add short "Licensing & Commercial Use" section (Core MIT + Pro/Enterprise)
  - [ ] Link prominently to website, docs, Discord, and GitHub Discussions
  - [ ] Create initial labels and a few "good first issue" tickets
  - [ ] Enable GitHub Discussions and define categories (Q&A, Ideas, Show & Tell)

- [ ] **Community setup**
  - [ ] Create AICO Discord server
  - [ ] Define base channels (#welcome, #announcements, #support, #backend, #frontend, #memory-and-kg, #showcase)
  - [ ] Write short community guidelines / code of conduct
  - [ ] Add Discord invite link to website, README, and docs

- [ ] **Cornerstone content**
  - [ ] Article 1: "Why I’m building a local-first AI companion (AICO)" (publish on `/aico` or blog)
  - [ ] Article 2: "Inside AICO’s architecture: message bus, memory, and modelservice"
  - [ ] Article 3: "Designing an AI that remembers you: AICO’s memory system"
  - [ ] Add CTAs to each article (GitHub + AICO Pro/Enterprise contact)

- [ ] **Branding & naming (interim)**
  - [ ] Use "AICO by Boeni Industries" consistently in website, README, and docs
  - [ ] Add legal footer / imprint references to "Boeni Industries AG" where appropriate (website, legal pages)
  - [ ] Perform a basic trademark/name search for "AICO" in key jurisdictions and note results

- [ ] **HN & Reddit launch**
