# Avatar Models Directory

## Copy Your GLB File Here

Please copy your `avatar.glb` file to this directory:

```bash
cp /path/to/your/avatar.glb /Users/mbo/Documents/dev/aico/frontend/assets/avatar/models/avatar.glb
```

## Expected File

- **Filename**: `avatar.glb`
- **Format**: GLB (binary GLTF)
- **Source**: Ready Player Me
- **Size**: Recommended 2-5MB

## Requirements

The GLB model should include:
- ✅ Blend shapes for facial animation (eyes for blinking, mouth for speaking)
- ✅ Skeleton/bones for body animation (Spine1/chest bone for breathing)
- ✅ Optimized for real-time rendering

## Verification

After copying, verify the file is here:
```bash
ls -lh /Users/mbo/Documents/dev/aico/frontend/assets/avatar/models/
```

You should see `avatar.glb` listed.
